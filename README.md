# Calculadora React
Calculadora feita em React.

## Tecnologias usadas
- React 18
- Create React App (react-scripts)
- JavaScript
- CSS
- Node.js e npm

## Como iniciar o projeto
Instale as dependencias e inicie:

```bash
npm install
npm start
```

## Estrutura do projeto
- src/: componentes e logica da calculadora
- public/: arquivos estaticos
